# seatAvailability

映画館のチケット販売時刻直前の席の埋まり具合を画像で保存するものです。  
自分が予約した映画の上映の際に、予約した席の周りがどれだけ埋まってるか(隣や前が気になる)を知るために。

- [ ] 作りかけというか作り始めです。
